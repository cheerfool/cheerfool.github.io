Group Leader, directed by Prof. Kenny Q. Zhu Jul.2013 ~ present

  • Construct a Chinese concept association network from three data sources including Chinese Wikipedia, Baidu Encyclopedia and Hudong Encyclopedia

Dynamic Advertisement and Video Recommendation Shanghai, PRC
Research Assistant, directed by Prof. Kenny Q. Zhu Mar.2013 ~ Jul.2013
  • Built a tool to automatically extract information for recommendation from the web
  • Built a basic Chinese word-relatedness knowledge base from Chinese Wikipedia, Baidu Encyclopedia and Hudong Encyclopedia
  • Designed a cross-domain recommendation system based on my previously built Chinese knowledge base
  • Built a video web site demo of the system, which can dynamically recommend videos and advertisements according to a user’s watching habits and features
Ultimate Goal
To build an innovative real-time cross-domain recommendation system, aiming at recommending relevant advertisements and video programs based on users' preferences and behaviors. (PredictTV)
System Overview
This recommendation system captures the real-time behaviors and feedbacks from audiences, and send the information back to our server. Then, based on users' current behaviors and history features, the backstage analysis module of our recommendation system updates USER-PREFERENCE model to offer more accurate personal service such as recommend TV shows and Internet news the user might be interested in. On the other hand, more accurate user model will be able to provide better Advertisements aimed at specific user.
